What ArchHub does

Six things no other AEC tool does in one chat.

✦

Chat drives every tool

"Dimension all walls" runs in Revit. "Build this sketch" runs in Blender. Code generated, executed, returned — never pasted.

⌘

Skills you save and share

Useful chat → /skill save → JSON. ComfyUI-style copy-paste. Cloud-synced via your private GitHub.

◇

Vision input

Paste a sketch. Claude / GPT-4o / Gemini reads it, infers parameters, builds the mass.

⤳

Sketch → production

Six-stage chained Skill: extract mass → push to Speckle → set up Revit → build walls → place openings → generate sheets. One click.

◐

Multi-LLM, BYO key

OpenRouter (real OAuth) or Anthropic, OpenAI, Google direct. Local Ollama for IP-sensitive work. Your choice.

⏵

Click-only setup

Installer with desktop icon, in-app auto-update, first-run wizard. No terminal, ever.
