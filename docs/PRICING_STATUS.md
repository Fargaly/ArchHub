# Pricing Status (Canonical)

_Last updated: 2026-05-07._

This file is the canonical source for user-facing pricing state.

## Current status

ArchHub is currently in **open beta** and **all shipped features are free**.

- No credit card required.
- No paywall on currently shipped features.
- Pro/Studio/Enterprise plan names may appear in roadmap/strategy docs, but those tiers are **not fully active billing plans yet**.

## Planned tiers (not fully active yet)

- Free
- Pro
- Studio
- Enterprise

Planned tier details are subject to change until billing is enabled.

## Rules for docs

When editing user-facing docs (`README.md`, `QUICKSTART.md`, `docs/index.html`), align all pricing language with this file.
