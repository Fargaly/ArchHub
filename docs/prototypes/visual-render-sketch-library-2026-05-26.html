Visual · Render · Sketch — unified library category

id: AgDR-0049 (proposed) · session: post-workshop · founder asked "where is the huge extended library" · builds-on: AgDR-0014 + AgDR-0040

Why this exists

"I think previously i signed off a huge extended library and catalogue for modular and logical workflows incorporating image generation · sketches · view ports · etc."
— founder, 2026-05-26

Image-generation, viewports, and sketches exist today but scattered across ai.*, connector.run → ComfyUI/Magnific/Photoshop, and per-host revit.create_view / acad.layout_* ops. There is no top-level visual category in LM_NODE_GRAMMAR. A user searching for "render" finds nothing under one roof. This prototype adds a unified VISUAL category with four sub-rails: render · image-gen · sketch · viewport.

How a workflow looks (composer mock)

Composer prompt: "hero render of the lobby from the southwest, photoreal, with my Missoni cladding swatch" 

view.create(view_family="3D", camera="SW iso", target=Lobby)
  ↓
view.thumbnail(view=$above, w=1024, h=768)              ← Revit hands JPG
  ↓
img.material_swatch(palette="Missoni", count=3)         ← ComfyUI side-rail
  ↓
img.image_to_image(
  src=$thumbnail,
  refs=$swatches,
  prompt="photoreal, golden hour, Missoni cladding"
)
  ↓
img.upscale(model="magnific-creative", factor=2)         ← final hero render

What does NOT change

  • No new connector. ComfyUI, Magnific, Stable Diffusion, Revit, Rhino, AutoCAD already on the connector list. img.* + view.* compose those.
  • No engine surgery. All nodes are impl.kind=graph (per AgDR-0040) — they're just typed wrappers around existing ops.
  • Wires stay Speckle. Per AgDR-0012 / AgDR-0016 / AgDR-0048 the substrate is unchanged.
  • Approval gates inherited. Every host_write node still goes through Plan / Auto / YOLO per USER-AGENCY MANDATE.
