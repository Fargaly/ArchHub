Nodes built from
modular logic.

A node is not a data envelope around a code blob. A node is a graph — its logic composed from the same modular elements everything else is. Open a node, find more nodes. The AI manipulates logic the way it manipulates a workflow: by wiring. This page is the design — look, then pick the three forks at the bottom.

Fork 3 — where does a composite node’s typed I/O come from?
Option ARecommended
Auto — derived from the inner graph
The composite’s ports ARE the open ends of its inner graph. Wire the inside; the outside contract appears. Nothing to keep in sync. One truth. (How Blender node groups work.)
ports light up from the inside ▸ outer face
Option B
Declared — author writes the contract
The I/O is a hand-written spec; the inner graph must satisfy it. More control + a stable public face, but two things to keep aligned. (How a Houdini HDA parameter interface works.)
contract declared first ▸ inner graph must match

4 · Prior art — “graph packaged as a node” is proven

Every serious node system reached the same answer. None ship a code-blob node as the primary unit.

Houdini — HDA · VFX
A subnetwork of nodes packaged as one reusable, nestable asset with a declared parameter interface.
Blender — Node Groups · 3D
Select nodes → group. I/O auto-derived from the Group Input/Output you expose. Nestable.
Unreal — Blueprints · games
Functions, Macros, Collapsed Graphs — a graph collapses into a callable node. Logic is the graph.
LabVIEW — subVI · instrumentation
Any VI used inside another. The connector pane wires the typed I/O. Composition is the whole language.
n8n — Sub-workflows · automation
A workflow called as a single node inside another. Exactly the recursion.
ComfyUI — Group Nodes · AI
Bundle a sub-graph into one node. The community’s answer to per-node sprawl.

5 · What this buys

Manipulate — rewire one inner node, never rewrite a blob.  ·  Scale — every logic-fragment is itself a library node, reused.  ·  Inspect — typed + diffable, zero opacity.  ·  One mental model — the AI (and the user) composes logic the same way it composes a workflow. Wiring. That is the whole skill.
