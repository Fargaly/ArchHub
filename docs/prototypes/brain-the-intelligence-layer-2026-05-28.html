Worked example — one sentence in, real plan out

You say: "clean up the repo without breaking anything, commit it all."

Brain pulls
  • Memory: "no waits >60s", protected files list
  • Mandate: ANTI-LIE, PROTOTYPE-FIRST
  • Skill: repo-overhaul (4-scout audit)
  • Setup: git hooks, brain daemon
Brain flags
  • challenge protected .cs files — needs env flag
  • challenge don't commit secrets
  • time ~20 min for safe queue
AI executes
  • Audit → commit by track
  • Run tests, verify green
  • reversible per-track commits
  • Show you the result
Every result writes back into Memory + mints new Skills → the brain is sharper on your next sentence. This is the moat.
