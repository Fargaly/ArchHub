What ArchHub does

Six things no other AEC tool does in one chat.

✦

Chat drives every tool

"Dimension all walls" runs in Revit. "Build this sketch" runs in Blender. Code generated, executed, returned — never pasted.

⌘

Skills you save and share

Useful chat → /skill save → JSON. ComfyUI-style copy-paste. Cloud-synced via your private GitHub.

◇

Vision input

Paste a sketch. Claude / GPT-4o / Gemini reads it, infers parameters, builds the mass.

⤳

Sketch → production

Six-stage chained Skill: extract mass → push to Speckle → set up Revit → build walls → place openings → generate sheets. One click.

◐

Multi-LLM, BYO key

OpenRouter (real OAuth) or Anthropic, OpenAI, Google direct. Local Ollama for IP-sensitive work. Your choice.

⏵

Click-only setup

Installer with desktop icon, in-app auto-update, first-run wizard. No terminal, ever.

What makes ArchHub different

Six commitments built into the product — none of them optional.

⌘

Open source you can audit

Every line of the desktop app is MIT-licensed and on GitHub. Fork it, read it, run it offline. Your firm's IT team can verify what the binary does before it touches a project file.

◇

Local LLM option for IP-sensitive work

Plug in Ollama and the entire chat runs on your machine — no model traffic leaves the laptop. Switch to a cloud provider only when you choose to.

✦

Skills you own and edit

Every saved Skill is a JSON file. Copy, paste, version, fork, delete. Nobody else's marketplace, nobody's lock-in.

⤳

Multi-tool, multi-LLM, multi-host

Revit, Blender, AutoCAD, 3ds Max, Speckle — driven from one chat. Anthropic, OpenAI, Google, OpenRouter, local — your choice of model. No silos.

◐

BYO key, BYO firm relay

Bring your own API keys directly, or point ArchHub at a self-hosted firm relay so provider keys never sit on architects' laptops. Both paths are first-class.

⏵

Free tier with no credit card

Download, install, use. No trial countdown, no upsell modal, no payment-method required. Pro and Studio exist for teams who want sync and a managed relay — both ship with v1.0.
