# ArchHub trademark filing — USPTO TEAS Plus submission pack

**Purpose:** everything you need to file the ArchHub trademark with the
USPTO yourself in under 60 minutes. The forms ask the questions; this
doc gives the answers.

> ⚠️ This is a prep doc, not legal advice. For complex prior-art or
> multi-class strategy questions, consult a TM attorney (~$300–$600
> one-time). For a simple software / SaaS mark like ours, TEAS Plus
> + this guide is sufficient for most founders.

## Marks to file — **minimum viable filing** ($350)

strategy down to a **$350 single-filing** strategy. Rationale: the
wordmark in a single class gives 90% of the protection at 25% of the
cost, and we can layer the stylized mark / additional class on later
without losing priority.

**Total today: $350.** (Down from $1,400.)

### Why TEAS Standard, not TEAS Plus

TEAS Plus saves $100 but requires you to use pre-approved descriptions
from the USPTO ID Manual *exactly*. The Class 042 ID for "SaaS in AEC"
isn't a clean Plus match. TEAS Standard ($350) lets you write a custom
description — important for our specific AEC + AI niche.

### Why Class 042 (not 009)

Class 042 covers "providing online non-downloadable software" — our
**cloud** revenue lives here (Solo, Studio, Firm tiers via Stripe).
Class 009 covers "downloadable software" — useful when desktop is the
business, but our model is open-core: desktop is free + cloud is paid.
Protect where the revenue is.

When desktop revenue meaningfully grows (custom enterprise distribution,
white-label, etc.), file Class 009 in Year 2.

### What ™ vs ® gets you today

Until USPTO registration issues (~9-14 months), use the **™** symbol.
Common-law trademark protection applies in every US state where you've
actually sold the product. After registration, use **®** — that flips
nationwide protection on, plus the right to:
- Recover statutory damages ($25k-$200k) without proving actual harm
- Treble damages for willful infringement
- Use TM in customs to block knock-offs at the border
- File in USPTO TMEP database — visible to anyone searching

### International Class 009 — Software products
**Class header:** Computer software; downloadable software; recorded software.

**Goods & Services description** (use ID Manual entry to qualify for TEAS Plus):

> Downloadable computer software for use in architecture, engineering,
> and construction (AEC), namely, software for automating design
> workflows, integrating computer-aided design (CAD) applications with
> artificial intelligence services, managing parametric design
> parameters across multiple software hosts, and generating
> documentation from building information modeling (BIM) data.

**ID Manual reference:** Use entry **009-0247** "Downloadable computer
software for [specify function]" — TEAS Plus accepts this with our
custom function string above. Verify the entry is still listed at
[idm.uspto.gov](https://idm.uspto.gov/) before submitting.

### International Class 042 — SaaS / Cloud services
**Class header:** Scientific and technological services; computer programming.

**Goods & Services description:**

> Software as a service (SaaS) services featuring software for use in
> architecture, engineering, and construction (AEC), namely, providing
> online non-downloadable software for automating design workflows,
> integrating computer-aided design (CAD) applications with artificial
> intelligence services, and synchronizing project data across
> distributed teams.

**ID Manual reference:** Use entry **042-0136** "Software as a service
(SAAS) services featuring software for [specify function]".

## After filing — what to expect

| Week | Event |
|---|---|
| 0 | File the application. Serial number issued immediately. |
| 1–4 | Application appears in TESS as "live application, pending exam" |
| 12–16 | Examining attorney assigned, first action issued |
| Best case | Approval → Notice of Publication → Opposition window (30 days) → Registration |
| Worst case | Office Action (refusal); you respond, possibly with an attorney |
| Total time | 9–14 months end to end if no opposition |

## International strategy

After US registration is allowed (or even at filing), use the Madrid
Protocol to extend to other jurisdictions through a single application
filed in the US. Priority targets:

| Jurisdiction | Reason | Approx fee (Madrid) |
|---|---|---|
| European Union (via WIPO) | Largest AEC market outside US | $1,000–$1,500 |
| United Kingdom | Big BIM consultancy market | $500 |
| United Arab Emirates | Home market; AEC capital of MENA | $1,200 (national filing) |
| Canada | Adjacent to US AEC firms | $700 |

Madrid filings are deferred until **after** the US wordmark is
allowed — you need that registration as the "base mark."

## What not to do

- ❌ Don't file under "intent to use" if you're already shipping —
  changes the timeline + cost
- ❌ Don't skip the TM search — a conflict at exam stage costs more to
  resolve than to avoid
- ❌ Don't use a generic "computer software" description — TEAS Plus
  rejects vague descriptions; specificity gets you the lower fee
- ❌ Don't file under your personal name if you've already formed an
  LLC — assignment fees later will hurt
