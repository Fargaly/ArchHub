# ArchHub status audit — 2026-05-14 (v1.4.0-alpha)

Honest table of what works end-to-end vs what's still decoration. Founder
directive: "nothing for show".

**v1.4 state snapshot**
- **18 host families** detected (was 6 in v1.0 prototype): Revit /
  AutoCAD / 3ds Max + Blender / Rhino / Speckle + Outlook / Teams /
  Notion + LM Studio / Antigravity + Photoshop / Illustrator / InDesign
  + Word / Excel / PowerPoint + Dropbox.
- **115+ bridge slots** in `app/bridge.py` (was ~80) — sessions / hosts
  / models / chat / memory / skills / permissions / providers / graph /
  wires / node-MCP / subgraph / composer / agent / theme / storage.
- **AI-agent composer** (`bridge.agent_step`) — natural-language intent
  ("ping outlook") → spawns host node + conversation node + wires them
  via a 7-tool schema (spawn_host / spawn_node / wire / focus /
  rename / delete / run).
- **Native PyQt `SettingsDialog`** — 5 real tabs (Providers / Memory /
  Profile / Storage / Shortcuts) replacing the decorative 10-tab JSX
  preview overlay.
- **Node Library** — most-used row at top, collapse-all / expand-all
  toggle, search box, drag-to-canvas.
- **Composer NL parser** — intent extraction lifts `"ping outlook"` →
  `[spawn h_outlook, spawn i_conv, wire status→conv.system]`.
- **Session = canvas** — one session is one persistent autosaved
  canvas; switching swaps the workspace.
- **Wire engine v2** — 28 px snap radius, hover preview glow (green
  ok / red refused), drop-on-node-body auto-port-pick, refusal toast
  with reason.
- **Studio v2 design** — `studio-lm.jsx` is canonical; the alt-design
  prototypes (`atelier.jsx`, `cockpit.jsx`, `blueprint.jsx`,
  `studio-vision.jsx`, etc.) are reference-only in
  `/tmp/archhub-handoff/`.
- **Fly cloud destroyed** — `archhub-cloud.fly.dev` and
  `archhub-agents.fly.dev` both decommissioned. Zero monthly spend.
  Cloud features fall back to local-only or BYO-relay until founder
  greenlights a managed plane.
- **Revit add-in deployed** to Revit 2020 / 2023 / 2024 / 2025
  AddIns folders.

Legend
- ✅ functional + tested
- ⚠ functional but partial (works in happy path, gaps documented)
- ❌ decoration / stub
- 🛠 in flight (agent currently running)
- ⏸ blocked on user action

3. **Right-rail param inspector empty** — ✅ NodeRail+FullParam editable for text/number/boolean/enum/version/document config fields.
4. **Browser context menu pollution** — ✅ `web_shell.py` line 92 NoContextMenu.
5. **Delete key may not work in some focus states** — ✅ window+document capture-phase listener handles Delete/Backspace globally; only suppresses inside INPUT/TEXTAREA/SELECT/contentEditable.
6. **Rich node right-click menu** — ✅ NodeMenu (line 2554) — Run/Freeze/Rename/Duplicate/Save-as-Skill/Expand-Subgraph/Disconnect-all/Delete/Properties.
7. **Save workflow as Skill (subgraph)** — ✅ Cmd-G compose_subgraph + bridge.save_as_skill.
8. **Composer `/wire` `/freeze` `/delete` `/rename` `/duplicate` `/properties` `/connect` `/disconnect` commands** — ✅ bridge.parse_composer_command + `lm-composer-action` dispatch on canvas.
9. **SettingsTheme / Profile / Shortcuts / Storage** — ⏸ preview-only overlay (never opens in desktop mode — `bridge.open_settings` routes to native PyQt dialog). Not user-visible. Wiring deferred until/unless founder needs them in preview build.
10. **Host-pill row in canvas header** — ✅ HostPillRow component pulls `bridge.get_all_hosts` every 25s (matches detector TTL). Click pill → spawns host node via `lm-spawn-host` event.
11. **Auto-register every mounted node as MCP server** — ✅ NodeRenderer useEffect calls `bridge.register_node_mcp(id, envelope)` on mount, `unregister_node_mcp(id)` on unmount.
12. **Chat TypeError "LLMRouter.complete() got unexpected keyword argument 'prompt'"** — ✅ FIXED 2026-05-14. Root cause: `bridge.send_chat` fallback path called `router.complete(prompt=, conversation=)` but signature is `(history, model, on_chunk, on_tool_invocation)`. Rewrote send_chat to use the chat-widget path primarily; added `send_chat_history(session_id, text, history_json)` slot for JSX to pass the focused conv node's full message history. Streams chunks via on_chunk → chat_chunk signal.
13. **Wire mechanism overhaul** — ✅ snap radius 28px (Houdini/UE5 magnet feel), hover preview glow (green=ok, red=refused), Blender-Geo-Nodes-style drop-on-node-body auto-port-pick, toast banner with refusal reason (incompatible types / would-cycle / dupe / self), drop-on-empty-canvas fires `lm-wire-promote` event for future create-node palette.
14. **Right-rail duplicate composer + fake "FROM THIS CONVERSATION" mock** — ✅ removed. ConversationRail now shows scrollback only + live `node.tool_trace` (collapses to nothing when empty). FloatingComposer at bottom is the single source of truth.
