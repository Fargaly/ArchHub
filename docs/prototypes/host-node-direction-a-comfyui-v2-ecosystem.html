Host node v2 — Direction A + ComfyUI hover-promote in the ecosystem you already built

Founder pushback (2026-05-21): "Think further — design can be better. And remember this doesn't wipe out the work already done; make sure it's incorporated." This v2 prototype answers both: the host node is redesigned around four explicit mechanics + shown LIVE inside a graph that re-uses every shipped slice — typed primitives, groups + collapse + nesting, Save-as-Skill, ai.plan, Adapter, Share, Code typed-split, Floating disable verbs, library-first search.

Four mechanics drive this node.
  1. Op grid + active tile (Direction A) — all 13 ops visible; active op expands inline. No dropdowns.
  2. MAIN ⇄ ADVANCED I/O — primary sockets always visible; advanced (everything else) collapsed by default; user hovers + promotes per row.
  3. Hover-revealed sockets on left edge (input promote) AND right edge (output pluck), with typed-pill hint. ComfyUI's Convert-Widget-to-Input gesture in its richest form.
  4. Floating settings bar for run-mode + disable verbs (pin / freeze / bypass / preview-off — AgDR-0002 reuse).
What this prototype keeps from prior shipped work (no rewrites needed):
  • Slice A 16 per-host master nodes — per-host brand stripe (Revit orange · AutoCAD red · 3ds Max purple · Excel cyan).
  • Slice B / AgDR-0002 disable verbs (pin / freeze / bypass / preview-off) — shown in the floating bar + flag dots.
  • Slice C / C2 / C3 Groups + collapse-to-giant-node + nesting — Stage 4 below shows this host node INSIDE a collapsed group with boundary-promoted sockets per AgDR-0005's algorithm.
  • Slice D / AgDR-0007 typed wire colours (purple = element, blue = list, yellow = number, cyan = string, green = bool) — every wire in this prototype follows the palette.
  • Slice G / G2 / AgDR-0010 Save-as-Skill preserves promoted sockets — the skill wrapper's I/O equals the host node's rail.
  • Slice H/I/J/K + AgDR-0019/0020 typed primitives — code_expr, ai_chat, ai_plan, constants, all coexist in Stage 3's graph.
  • M1.5 SHARE + M2 ADAPTER + AgDR-0016/0017/0018 share.publish + adapter.cad_to_revit_wall + revit.send_to_speckle all consume the host node's promoted outputs.
  • M4 / AgDR-0021 ai.plan reads the host node's promoted-socket schema as its tool surface.
  • AgDR-0013 LIBRARY-FIRST gate — Cmd-K popup shown in Stage 3 surfaces existing matches before suggesting new node creation.
  • AgDR-0015 a11y + contrast tokens — every colour referenced here lives in the LM palette + passes the WCAG audit.
