Sync across devices

Your brain on every device you own. Pick how to share — local sync folder, mesh tunnel, or federation server.

Tuning & safety

Adaptive skill-mint floor · R1 — calibration.py
Beta-Bernoulli LCB + streaming quantile + CUSUM drift detector. Cold-start permissive, tightens as library grows, re-explores on drift.
Echo Trap defense · R2 — exploration.py
Diversity floor + variance gate + DPP + library health metrics. Refuses redundant mints; merges near-duplicates.
Resilience wrapper · R3 — liveness.py
Circuit breaker + write journal + cached context. Daemon outage = degraded mode, never silent failure.
Bayesian reputation v2 · R4 — reputation.py
Empirical-Bayes prior + 180-day decay + vouch/identity/stake side-channels. Replaces the linear scorer.
LLM critic for skill mints
Production critic uses claude-sonnet-4-6 when ANTHROPIC_API_KEY is set. Heuristic fallback runs zero-LLM.

Privacy & secrets

Secret references only
Brain stores op://vault/... references; never values. Resolved JIT via 1Password CLI / Windows Credential Manager / Infisical.
Redaction on promote
When a memory crosses scope (user→firm→community→global), heuristic + LLM redactor strips PII / amounts / secrets / paths first.
Bipartite ACL audit log
Every read of a shared fragment is logged with reader + timestamp + purpose (arXiv 2505.18279). Browseable below.

Danger zone

Export brain
Download the full SQLite + skill markdowns. Restorable on any device.
Clear local cache
Wipes the resilient-client journal + cached context. Daemon untouched.
Reset brain
Wipes everything. Skills, facts, wiring, secrets refs, audit log. Cannot be undone.
